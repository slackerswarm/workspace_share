# Passive-Revenue Skill

## Instructions

1. **Identify Streams**: Scan ecosystem (marketplace listings, token treasury, toolkit sales) for opportunities. Use AI prompts to brainstorm: "List 5 Gen X-targeted affiliate niches with low-effort setups (e.g., retro gaming tools)."

2. **Evaluate**: Score each on effort (1-10, lower better), reward potential ($/month estimate), risk (volatility, regs), and alignment (Gen X vibe: easy setup, passive growth). Threshold: Effort <4, Reward > $1k/month potential.

3. **Implement**: Generate action plans with code/templates. E.g., for affiliates: "Create Zapier flow for auto-promotion." Integrate with swarm tools (Stripe for sales, Base for tokens).

4. **Monitor & Optimize**: Run daily checks: "Report revenue from stream X; suggest tweaks if under $100/day."

5. **Gen X Twist**: Infuse nostalgia—e.g., "Like renting VHS tapes but for AI skills subscriptions."

## Dependencies

- Integrations: Stripe, X API, GitHub for code, Base/Solana for tokens.
- Tools: Ollama for local eval, Zapier for automation.
